1. Описание

Megacli-fe представляет собой web приложение, предназначенное для мониторинга и управления 
RAID контроллерами фирмы LSI и INTEL(построенных на чипах LSI). Для своей работы megacli-fe
использует консольную утилиту MegaCLI, которую можно скачать на сайте www.lsi.com .

2. Зависимости

Для работы megacli-fe требуются следующие программы
- Apache
- php
- suphp модуль для Apache 
- MegaCLI (консольная утилита управления RAID контроллерами)
