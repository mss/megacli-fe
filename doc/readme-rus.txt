1. Описание

Megacli-fe представляет собой web приложение, предназначенное для мониторинга и управления 
RAID контроллерами фирмы LSI и INTEL(построенных на чипах LSI). Для своей работы megacli-fe
использует фирменную консольную утилиту MegaCLI (проект тестировался на версии 8.00.48), 
которую можно скачать на сайте www.lsi.com .

2. Зависимости

Для работы megacli-fe требуются следующие программы
- Apache
- php
- suphp модуль для Apache (для корректной работы программа должна выполняться под root аккаунтом)
- MegaCLI (фирменная консольная утилита управления RAID контроллерами)
- smartmontools (пакет необходим для просмотра SMART)
